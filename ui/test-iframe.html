Datadog Iframe Authentication Test

This page simulates how your Support UI would appear when embedded in a Datadog dashboard.

Embedded Support Dashboard (Iframe)

How to Test:

  1. Make sure your Next.js app is running on http://localhost:3000
  2. The iframe above should show the login button (since it's unauthenticated)
  3. Click "Connect with Azure AD" or "Connect with Google"
  4. A popup window should open for authentication
  5. Complete the authentication in the popup
  6. The popup should close and the iframe should refresh with authenticated content

Notes:

  • The iframe is sandboxed to simulate Datadog's security restrictions
  • Cookies are configured with SameSite=None; Secure for cross-site scenarios
  • CSP headers allow framing from Datadog domains
